---
title: Scroll columns
date: March 24, 2023
---

Set the max height and overflow styles of [Bootstrap](https://getbootstrap.com) columns to independently scroll the columns. This isn't JavaScript, but it's related to website layout so it might be useful for designing web applications.

<img src="../images/scroll-columns.png" style="max-width:100%;" alt="scroll columns">
